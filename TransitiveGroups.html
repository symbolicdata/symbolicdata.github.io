Transitive Groups

About

These data refer to classificational data of groups collected in the Database for Number Fields of Jürgen Klüners & Gunter Malle. This transformation is a very first attempt to compile fingerprints from this collection of transitive groups.

The fingerprints were directly compiled from the corresponding URL in the Database for Number Fields by Andreas Nareike in March 2013.

The transformation code is available from our maintenance repo.

Structure of the Data

  • The Transitive Groups Knowledge Base for download and as Linked Data
  • The Resources are available directly from the Database for Number Fields.
  • TransitiveGroups.Ontology - an informal description of the Ontology of the Transitive Groups Knowledge Base developed so far.
